Java流

字符流与字节流

Java IO中的流操作分为两类:

  • 字节流(byte)：主要是:InputStream（输入） 和 OutputStream（输出）
  • 字符流：操作字符类型数据，一个字符占两个字节。主要是Reader（输入）和Writer（输出）
    流操作一般流程:

  • 字节流主要操作字节类型数据（byte）；字符流主要操作字符类型数据，一个字符占两个字节

  • 字节流在操作时本身不会用到缓冲区（内存），而是对文件本身直接操作的；字符流在操作时使用了缓冲区，通过缓冲区再操作文件
